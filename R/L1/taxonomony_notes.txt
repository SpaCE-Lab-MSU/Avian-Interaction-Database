## About this file

This is a scratch file for keeping notes about discoveries for taxonomic names
that need to be updated.   This started from pulling comments out of the original
AvianInteractionsData_L1 script from PLZ December 2024-May 2025, but adding outputs
from the Clements checklist and our Names list.  Can also copy/paste out of
Avibase as needed.

these notes are meant to help complete the columns in  the manual_taxonomics_resolutions.csv file


summmary, moving this manual resolutions/overrides as :

  original R code -> these notes -> resolutions CSV table

and the workflow will read the table and apply the changes.  If all changes are
discussed in these notes are accounted for in the resolutions table CSV file,
then they can be delete from this notes file (otherwise will be duplicated/confusing
which is the authority)
